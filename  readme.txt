A simple and light-weight plugin for sending email notification on reply to a comment.It uses the basic wp_mail function.

== Description ==
Users can choose through check box whether receive email notifications if someone replies to their comment:
By default the check box is checked.

## Requirements
* Wordpress 2.9.2 or higher
* PHP 5.1 or higher
* MySQL 5.x or higher
* Threaded/Nested comments option Enabled

## Main Features
*very Simple to use
*Bug free
*compatible with other similar plugins


== Installation ==

1. Upload the entire folder and all the subfolders to your Wordpress plugins folder
2. Activate it
